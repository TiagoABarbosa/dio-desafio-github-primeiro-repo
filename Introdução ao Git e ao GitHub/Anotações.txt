Sobre o Git:
Git é um sistema de controle de versões distribuído, usado principalmente no desenvolvimento de software


Link para download do Git: https://git-scm.com/downloads
O Git Bash é um terminal extendido para otimizar o uso do Git.

Link para download do Typora: https://typora.io/
Typora é uma ferramenta projetada para facilitar o código-fonte de remarcação para escritores e leitores.
